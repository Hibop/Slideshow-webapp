Weinre调试服务器 ⤴

  1. 安　装：npm -g install weinre
  2. 移动端和调试服务器连入同一个wifi
  3. 服务器：开启weinre监听端口
  4. weinre --httpPort 8081 --boundHost 192.168.0.106
  5. 移动端：在页面中引入target-script-min.js
  6. 服务器：打开Debug页面，进入相应target
  1. 局限性
  2. 不支持实时改CSS
  3. 不支持单点调试
  4. 不支持非原生代码
